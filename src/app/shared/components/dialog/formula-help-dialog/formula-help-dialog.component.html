Formula help!
Note: Table is acessible using $Table, and it properties like : $Table.firstName
  • String : (Exa. table has two fields firstName and lastName, and we need both together) : {{ "$Table.firstName + $Table.lastName" }}
  • Number : (Exa. table has two fields quantity and price, and we need multiplication of both) : {{ "$Table.quantity * $Table.price" }} Note: Need to ensure both fields are of type number
  • Date : (Exa. we need to check if the record was created today or before ) {{ "if(new Date($Table.createdAt) > new Date()){ true }" }} Note: To convert date into JavaScript date object, we need to wrap it inside Javascript's new Date() For more info. you can check here
  • Radio : (Exa. table has a field called age and we need to return true if value is above 18) : {{ "if($Table.age > 18){ true }" }} Note: we don't use return statement inside formula
  • How to use Lookup fields: Image we have a lookup name 'plan' inside Orders table. We need to fetch plan name inside formula. We can do like this : $Table.plan.planName Note: planName field should be selected inside plan lookup field
  • How to use condition logic : Condition logic is most powerful feature of Formula. We can use JavaScript code here like so : {{ "if($Table.orderTotal > 500 && $Table.orderStatus == 'confirmed') { true; }" }} This will return true which can be used to confirm if the order was premium or anything related.
